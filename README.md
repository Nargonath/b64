## B64

Base64 encode and decode UTF-8 strings for node.js

## Instalation

1. Install [npm][1]
2. `npm install b64`

## Usage

      $ node
      var base64 = require('b64');
      var encoded = base64.encode("node.js");
      console.log(base64.decode(encoded));
      > node.js
